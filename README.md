# Negozi De Fummo

A minimal static landing page for **Negozi De Fummo** inspired by the provided screenshots.

## 🚀 How to run

1. Open `index.html` in your browser.
2. Approve the age verification prompt to see the site.

### Pages
- `index.html` – landing / shop showcase
- `login.html` – user login form
- `signup.html` – account registration form
- `admin-login.html` – demo admin login (use `admin@store.com` / `admin123`)
- `admin-dashboard.html` – admin dashboard view

## 🧩 Structure

- `styles.css` – layout + styling (responsive)
- `script.js` – age gate logic (localStorage)
- `auth.js` – login/signup/admin navigation logic

## ✨ Notes

- The site uses placeholder images from Unsplash.
- The age verification state is stored in `localStorage` so returning visitors won’t be prompted again.
